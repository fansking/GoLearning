# 为什么要学Go

第一次编辑于202.3.22，目前go语言国际形式不温不火，但在中国，go语言已有星星之火的势头，许多大厂诸如字节跳动、鹅厂、京东和b站等等已经或正在转为go语言。虽然目前来看，国内大部分公司还是JAVA,许多小厂因为生态的原因还是选择JAVA，但可以预见的是，Go现在的坑位还不算多，但随着其星火的蔓延，Go的缺口会越来越大，现在尽早占坑位那么未来越有优势（赌狗心态，压一手宝）。
